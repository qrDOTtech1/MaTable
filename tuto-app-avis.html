MaTable Pro - Guide d'utilisation

L'App Avis Google & Réputation IA

Comment configurer et utiliser le système de récolte d'avis intelligents pour propulser votre restaurant sur Google Maps.

1

Configuration dans le Dashboard

Tout commence dans votre espace administrateur. Rendez-vous dans le menu Avis des clients puis cliquez sur l'onglet ⚙️ Paramètres IA.

  • Le Lien Google : Collez le lien exact vers votre page Google My Business (le lien "Demander des avis" que Google vous fournit).
  • La Récompense : (Optionnel) Cochez la case pour offrir un code promo à vos clients s'ils laissent un avis (ex: "Un café offert", Code "CAFE2026").
  • Cliquez sur Enregistrer la campagne.

Aperçu du panneau de configuration

2

Le QR Code Magique

Une fois votre lien configuré, l'application génère automatiquement un QR Code unique pour votre établissement.

Où le placer ?

  • Sur les chevalets de table en fin de repas.
  • Imprimé sur vos tickets de caisse.
  • Présenté par le serveur via sa propre tablette/téléphone (Portail Serveur).

💡 Astuce : Cliquez sur "🖨️ Imprimer ce QR Code" directement depuis le tableau de bord.

3

L'Expérience Client (Le Parcours)

Le client scanne le QR code avec son téléphone. Voici ce qu'il se passe, étape par étape :

🧑‍🍳

1. Le Serveur

Le client sélectionne la personne qui s'est occupée de lui (avec sa photo). Ça personnalise l'expérience.

⭐

2. Les 4 Notes

Il donne une note sur 5 pour la cuisine, le service, l'ambiance et le rapport qualité/prix.

🤖

3. Le Chat IA

Magique ! L'IA pose 2 ou 3 questions simples au client (ex: "Qu'avez-vous mangé ?", "Un mot sur l'ambiance ?").

📝

4. L'Avis Parfait

L'IA génère un texte parfait. Le client n'a plus qu'à cliquer sur "Copier & Ouvrir Google" pour le coller.

💰

Bonus - Le Pourboire : Juste après la redirection vers Google, l'écran propose au client de laisser un pourboire par carte bleue (Apple Pay / Google Pay). S'il le fait, l'argent est crédité au serveur sélectionné à l'étape 1 !

4

Suivi et Résultats

De retour dans votre Dashboard, dans l'onglet "Avis des clients", vous pouvez observer les résultats de votre campagne :

⭐ Les textes générés

Vous verrez tous les brouillons générés par l'IA (les textes que les clients ont copiés), avec le détail des notes par critère et le nom du serveur concerné.

💝 Les pourboires reçus

Une liste claire des montants reçus via Stripe (Apple Pay/Google Pay), classés par serveur, pour que vous puissiez facilement redistribuer cet argent à vos équipes.
